# 📚 Apprentissage C#

Ce dépôt contient un projet d’apprentissage en C#.

L’objectif est de pratiquer les bases du langage et de découvrir progressivement le fonctionnement de l’écosystème .NET à travers différents exercices.

## 🎯 Contenu

- Exercices simples (variables, conditions, boucles)
- Manipulation de classes et objets
- Tests de logique métier
- Expérimentations diverses

## 🚀 Objectif

Ce projet n’a pas vocation à être une application finale, mais plutôt un espace de pratique pour :

- comprendre la syntaxe C#
- se familiariser avec la programmation orientée objet
- apprendre à structurer un projet

dotnet run
